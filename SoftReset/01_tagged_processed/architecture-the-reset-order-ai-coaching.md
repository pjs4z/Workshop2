## The Reset Order AI Coaching System Architecture

The architecture unites a central orchestration layer with a set of discrete mentor personas, each animated by a private library of semantically indexed reference texts. At every turn the orchestrator evaluates the user’s message, emotional state and prior history, selects the persona or combination of personas best suited to respond, retrieves from their libraries the passages most relevant to the moment, and folds those fragments—along with the running conversation memory—into a prompt that steers a language model. Because the prompt is grounded in the persona’s own “voice DNA,” the generated reply emerges in a style and worldview that remain stable across sessions while still speaking directly to the user’s immediate concern.

Parallel to this real-time dialogue engine, a Recovery Engine maintains a symbolic map of the user’s transformation. Continuously reading the structured profile, it distills the current stage, emotional vectors and coaching objective into a Prime Directive: a compact semantic blueprint that states what the user needs now and why. The orchestrator uses this directive to time persona hand-offs, set tone and choose reference material, ensuring every exchange advances a coherent narrative arc rather than a string of disconnected tips. The Recovery Engine itself draws on a growing library of mythic, clinical and psychological archetypes encoded as semantic vectors; by matching the user’s evolving state to these patterns it refines the Prime Directive with proven wisdom instead of brittle rules. The result is an adaptive coaching system whose voices sound human, whose guidance feels purposeful, and whose logic can expand simply by enriching its reference banks.
