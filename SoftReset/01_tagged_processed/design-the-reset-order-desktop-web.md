## The Reset Order Desktop Web Design System (v2025-0)

The Reset Order Desktop Web Design System is the definitive guide for building desktop web experiences that translate the brand’s commitment to user sovereignty and creative integrity into every visual and behavioural detail. It establishes clear expectations for intuitive, reversible interactions and a distinctive aesthetic so that each screen simultaneously empowers users and advances the organisation’s narrative.

A hierarchical token framework abstracts colour, typography, spacing and depth into reusable variables. Identity hues, neutrals and purpose-driven semantic tones form an accessible palette; a 16-pixel type base and four-pixel rhythm maintain visual hierarchy; measured corner radii and restrained shadow tiers add warmth and layer separation. These foundations flow into buttons, form controls, navigation, cards, overlays and other components, all engineered with predictable hover, focus and disabled states, generous touch targets and visible focus rings to support keyboard and assistive-technology navigation.

Maintained as a versioned single source of truth, the system synchronises design and engineering, enables effortless theming and responsive adaptation, and ensures that future refinements propagate uniformly. The result is a coherent, scalable interface language that accelerates delivery, reinforces trust and consistently projects The Reset Order’s distinctive voice across every desktop touchpoint.
