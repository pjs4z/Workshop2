# The Habit Loop

**Definition:** The four-step neurological feedback loop that allows the brain to automate solutions to recurring problems: Cue, Craving, Response, and Reward.

## Systems Dynamics & Relationships
This loop governs the formation and maintenance of all habits.
1. **Cue:** Triggers the brain to predict a reward ([[Prediction]]).
2. **Craving:** The motivational force; the desire to change one's internal state via a [[Dopamine-Driven Feedback Loop]].
3. **Response:** The actual habit performed (influenced by [[Law of Least Effort]]).
4. **Reward:** Satisfies the craving and teaches the brain to remember the step (influenced by [[Reinforcement]]).

- **Upstream (Driven by):** [[Environment Design]] (for Cues), [[Identity]] (for Cravings)
- **Downstream (Drives):** [[Automaticity]]

## Behavioral Engineering Context (COM-B)
- **Leverage Points:**
    1. Make it Obvious (Cue) via [[Implementation Intention]].
    2. Make it Attractive (Craving) via [[Temptation Bundling]].
    3. Make it Easy (Response) via [[Decisive Moment]].
    4. Make it Satisfying (Reward) via [[Habit Tracking]].
