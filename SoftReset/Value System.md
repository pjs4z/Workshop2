# Value System

**Definition:** A set of deeply held personal beliefs and priorities (such as health, family, or achievement) that serve as a counterweight to addictive impulses and a primary motivation for behavioral change.

## Systems Dynamics & Relationships
The engine of **The Virtuous Cycle of Natural Recovery**. It provides the reflective motivation to resist impulses.
- **Upstream (Driven by):** [[Community Reinforcement]] (social norms, family expectations).
- **Downstream (Drives):** [[Natural Recovery]]; reduced appeal of [[Addiction]]. When addiction takes over, the Value System is often degraded or ignored.

## Behavioral Engineering Context (COM-B)
- **Motivation:** Reflective Motivation (conscious decision-making based on priorities).
- **Leverage Points:** Values Alignment assessment to map conflicts between addiction and core beliefs (e.g., "Smoking violates my identity as a singer").
