# [[The Eisenhower Matrix]]
**Type:** #BFO/Independent_Continuant

## Concept Summary
The Eisenhower Matrix is an Independent Continuant in the form of an inscribed artifact (a document). It is an inert tool whose structure provides a normative framework for categorization. It does not act on its own but participates in the process of Taming the Biological Imperative by enabling the Human Agent to sort and prioritize tasks externally.
