# Cognitive Least Effort Loop

**Definition:** A reinforcing feedback loop where System 1 bypasses cognitive strain by substituting complex questions with simpler ones, thereby rewarding and reinforcing System 2's laziness.

## Systems Dynamics & Relationships
This loop describes the systemic interaction of the mind's dual-process architecture when faced with high cognitive load.
1. Faced with a computationally demanding [[Target Question]], [[System 1]] deploys the [[Mental Shotgun]].
2. This imprecise computation increases the availability of a simpler [[Heuristic Question]].
3. [[System 1]] uses [[Intensity Matching]] to fit the heuristic answer to the original target scale, catalyzing [[Substitution]].
4. [[Substitution]] generates an immediate, intuitive response that inhibits the critical vigilance of [[System 2]].
- **Upstream (Driven by):** High cognitive load / complex [[Target Question]]
- **Downstream (Drives):** Long-term reliance on [[System 1]] and suppression of [[System 2]] (Analytical Deliberation)

## Behavioral Engineering Context (COM-B)
Understanding this loop is critical for addressing the target behavior of Cognitive [[Substitution]]. Because the loop is naturally self-reinforcing (it effectively reduces mental effort, rewarding the brain's laziness), it must be artificially disrupted. 
- **Leverage Points:** Introduce deliberate cognitive speed bumps to disrupt cognitive ease. By forcing a pause (e.g., cooling-off periods, mandatory written justifications), the loop is broken, triggering the **Deliberative Monitoring Loop** where [[System 2]] acts as an enforcer rather than an apologist.
