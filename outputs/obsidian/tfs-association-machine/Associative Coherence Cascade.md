# Associative Coherence Cascade

**Definition:** A continuous, self-reinforcing feedback loop that generates a diverse and integrated pattern of simultaneous cognitive, emotional, and physical responses, wherein each activated element supports, strengthens, and confirms the others.

## Systems Dynamics & Relationships
This loop represents the holistic functioning of the associative machine. An initial stimulus causes [[System 1]] to trigger [[Associative Activation]] across nodes in [[Associative Memory]]. This rapid cascade evokes thoughts that involuntarily trigger physical responses via the [[Ideomotor Effect]]. Mediated by the biological hardware of [[Embodied Cognition]], these physical responses feed back into the mind through [[Reciprocal Priming]], which subsequently increases the activation of compatible ideas, locking the system into a state of [[Associative Coherence]].
- **Upstream (Driven by):** [[System 1]], [[Sociocultural Prime]]
- **Downstream (Drives):** [[Associative Coherence]], Automated behavioral output

## Behavioral Engineering Context (COM-B)
This reinforcing loop acts as the master **Motivation** mechanism for sustaining an affective state or behavioral momentum.
- **Leverage Point:** To shift user behavior, you do not need to persuade [[System 2]] rationally; instead, you only need to inject a single vector into the loop—either via a [[Sociocultural Prime]] (Opportunity) or a forced physical movement (Capability). Once injected, the cascade takes over, self-reinforcing the desired emotional or behavioral state (e.g., feeling amused, acting calm).
